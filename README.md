# hub-cn-web

一个用于归档与发布多个独立 HTML 页面的仓库。

## 项目简介

本仓库主要用于收集、整理和托管各类静态 HTML 页面。这些页面彼此独立，不依赖特定的后端服务或数据库，适合作为轻量级信息展示或功能演示的载体。

所有页面内容均以纯前端方式呈现，便于快速访问与离线保存。

## 目录说明

```
.
├── pages/          # 存放归档的 HTML 页面文件
├── assets/         # 页面引用的公共资源（如 CSS、JS、图片等）
├── index.html      # 仓库首页（可选，用于页面索引）
└── README.md       # 本说明文件
```

- `pages/`：每个独立页面为一个 `.html` 文件，命名清晰，便于识别。
- `assets/`：公共资源按类型或用途分目录存放，避免重复。

## 页面归档说明

- 每个 HTML 页面均为完整自包含文件（或引用 `assets/` 下的资源）。
- 页面内容符合基本 Web 标准，尽量兼容主流浏览器。
- 不包含跟踪脚本、广告或外部恶意资源。
- 归档时尽量保持原始结构与样式，必要时会进行格式调整或注释。

## 维护说明

- 欢迎提交 Issue 报告页面显示问题或提出改进建议。
- 本仓库不承诺对任何特定页面进行持续更新或内容修改。
- 提交 Pull Request 新增页面时，请确保：
  - 文件放在 `pages/` 目录下
  - 命名不包含特殊字符或空格
  - 不包含任何形式的推广或敏感信息

## 许可

本仓库内容仅供学习与参考使用，具体页面可能保留其原始版权信息。请合理使用。
